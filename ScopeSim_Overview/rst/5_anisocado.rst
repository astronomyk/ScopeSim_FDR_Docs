AnisoCADO

Documentation: https://anisocado.readthedocs.io/

Code Base: https://github.com/astronomyk/AnisoCADO

Continuous integration : https://travis-ci.org/github/astronomyk/AnisoCADO

Author: Kieran Leschinski
